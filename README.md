## 自述  

此模块是为了方便日常开发工作而整合的工具集，包括但不限于：
- 常用的导出函数
- 简单的事件聚合器
- 各类简化代码的扩展方法及数据模型
- [NLog](https://www.nuget.org/packages/NLog) 的包装调用
